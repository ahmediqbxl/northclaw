# NorthClaw

White-glove [OpenClaw](https://openclaw.com) deployment for Canadian startups. AI executive agents with Canadian data residency, PIPEDA compliance, and CAD pricing.

## Overview

NorthClaw is a managed service that deploys and hardens OpenClaw — an open-source AI assistant that runs 24/7 — for Canadian founders and executive teams. The service handles provisioning, security hardening, tool integrations, and ongoing maintenance so teams can get AI leverage without technical overhead.

## Tech Stack

- Static single-page site (`index.html`)
- No build tools or frameworks required
- Google Fonts (DM Serif Display, Syne, IBM Plex Mono)
- Vanilla JavaScript (FAQ accordion)

## Local Development

Open `index.html` in a browser. No build step needed.

## Deployment

Configured for [Netlify](https://netlify.com). Connect the repo and deploy — Netlify will serve `index.html` automatically.
